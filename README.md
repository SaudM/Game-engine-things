# 简介

《游戏引擎的那些事》是一本故事书，故事的主角是游戏引擎相关的人和事，其中包含游戏的商业故事，也包含了推动游戏与3D游戏引擎发展的人物故事。这是一本持续创作的书，也是一本众创书籍，但更重要的是，这里其实是个社区 —— 这里集聚着一批热爱游戏、热爱技术、热爱研究、热爱习作、热爱表达，并以此为乐的人。我们也欢迎您加入。书中的内容会编排为视频内容，会在YouTube，B站等视频平台同时发布。
