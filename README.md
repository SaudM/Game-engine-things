# 简介

《游戏引擎的那些事》是一本关于游戏引擎的书，这也是一本持续创作的书，一本众创书籍，但更重要的是，这里其实是个社群 —— 这里集聚着一批热爱游戏、热爱技术、热爱研究、热爱习作、热爱表达，并以此为乐的人。我们也欢迎您加入。书中的内容，会安排做成视频内容，发布在YouTube，B站等平台上。
